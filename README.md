# portfolio
# Harsh's AI Portfolio

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set environment variable: `GROQ_API_KEY=your_key_here`
3. Run: `streamlit run streamlit_app.py`

## Features
- Professional portfolio with dark theme
- AI chatbot powered by Groq Llama-3
- Responsive design
- Skills showcase
- Projects display

## Deployment
Deploy on Streamlit Cloud, Railway, or Render.
